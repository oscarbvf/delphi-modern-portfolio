# Delphi Modern Portfolio

This repository aggregates a set of modern Delphi projects designed to demonstrate
the evolution of the language from legacy versions (Delphi 5) to current Delphi releases.

## Goals

- Showcase modern Delphi language features
- Demonstrate real-world architectures
- Provide a clean and professional portfolio for international opportunities

## Projects

| Project | Description |
|-------|------------|
| HelloModernDelphi | Language features: Generics, RTTI, Anonymous Methods, Nullable |
| ServerHorse | REST API using Horse framework and SQLite |
| ClientVCL | VCL client consuming REST services |

## Topics Covered

- Modern Object Pascal
- Generics and RTTI
- REST APIs
- MVC / Clean Architecture
- Database access (FireDAC)
- Parallel Programming Library (PPL)
